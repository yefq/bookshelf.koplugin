# Bookshelf

A skeumorphic home screen for KOReader. Replaces the file manager on launch with a focused hero card for the currently-reading book and four chip-driven shelves (Recent / Latest / Series / Favourites).

> **v0.1 · in development.** The full feature set, install paths, and customisation cheatsheet land here when v0.1 ships.
